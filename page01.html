1. Conceitos Básicos do HTML

HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar conteúdo na web. Cada página da web que você vê é criada com HTML, que organiza o conteúdo em elementos distintos, como cabeçalhos, parágrafos, imagens e links.

Um documento HTML é composto por uma série de elementos, chamados de "tags". Essas tags são utilizadas para envolver o conteúdo e dar significado semântico à estrutura do documento.

<!DOCTYPE html>
<html>
  <head>
    <title>Exemplo de HTML</title>
  </head>
  <body>
    <h1>Olá, Mundo!</h1>
    <p>Este é um exemplo básico de uma página HTML.</p>
  </body>
</html>

O código acima é uma estrutura básica de HTML, onde o conteúdo é organizado dentro das tags de abertura e fechamento, começando com <html> e terminando com </html>.

2. Introdução ao HTML

HTML (HyperText Markup Language) é a linguagem padrão para a criação de páginas web. Ele define a estrutura do conteúdo em uma página web, utilizando uma série de elementos conhecidos como "tags".

Com HTML, você pode criar páginas estruturadas, incluindo cabeçalhos, parágrafos, listas, links, imagens e muito mais. Ele é a base de todo o desenvolvimento web.

3. Tags Básicas do HTML

As tags são os blocos de construção de uma página HTML. Elas são usadas para definir diferentes partes do conteúdo, como:

  • <h1> a <h6> - Cabeçalhos de diferentes tamanhos.
  • <p> - Define um parágrafo.
  • <a> - Cria um link para outra página ou recurso.
  • <img> - Insere uma imagem em sua página.
  • <div> - Define uma divisão ou uma seção.

Essas são algumas das tags mais comuns. Conforme avançamos, aprenderemos a combinar essas tags para criar layouts completos e funcionais.

4. Links e Imagens

Links e imagens são elementos essenciais em qualquer página da web.

O elemento <a> é usado para criar links:

<a href="https://www.exemplo.com">Visite o site</a>

Já o elemento <img> é usado para adicionar imagens à página:

<img src="imagem.jpg" alt="Descrição da Imagem">

É importante sempre usar o atributo alt para descrever a imagem, pois isso ajuda na acessibilidade e otimização para motores de busca.

5. Formulários em HTML

Formulários são utilizados para coletar dados dos usuários. Eles são criados com a tag <form> e podem incluir diferentes tipos de campos, como caixas de texto, botões de rádio, caixas de seleção, entre outros.

<form action="submit.php" method="post">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome">

    <label for="email">Email:</label>
    <input type="email" id="email" name="email">

    <button type="submit">Enviar</button>
</form>

Formulários são essenciais para interações dinâmicas em sites, como login, cadastro e envio de dados.

6. Estilos e Formatação

O HTML por si só define a estrutura do conteúdo, mas para tornar o visual mais atraente, utilizamos CSS (Cascading Style Sheets) para aplicar estilos e formatações. O CSS pode ser aplicado diretamente em uma tag, em uma seção <style> no cabeçalho, ou em arquivos CSS externos.

Veja um exemplo de como aplicar estilo diretamente no HTML:

<h1 style="color: blue; text-align: center;">Título Centralizado em Azul</h1>

8. Introdução ao CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas web. Ele permite alterar cores, fontes, layout, espaçamento, entre outros aspectos visuais de uma página. Juntamente com o HTML, o CSS dá vida ao design da web.

9. Conceitos Básicos do CSS

O CSS é baseado em regras que controlam a aparência dos elementos HTML. Cada regra é composta por um seletor e um bloco de declaração. O seletor indica qual elemento HTML será estilizado, enquanto o bloco de declaração define o que será aplicado a esse elemento.

10. Sintaxe do CSS

A sintaxe básica do CSS segue o formato:

seletor {
            propriedade: valor;
        }

Por exemplo, para alterar a cor de um texto em um parágrafo:

p {
            color: red;
        }

Isso aplicará a cor vermelha a todos os parágrafos (<p>) na página.

Você pode aplicar várias propriedades ao mesmo tempo:

p {
            color: red;
            font-size: 16px;
            text-align: center;
        }

Isso altera a cor do texto, o tamanho da fonte e alinha o texto ao centro.

11. Seletores no CSS

Os seletores no CSS determinam quais elementos HTML serão estilizados. Alguns exemplos de seletores comuns incluem:

  • Seletor de Tag: Aplica o estilo a todos os elementos de uma determinada tag. Exemplo: p
  • Seletor de Classe: Aplica o estilo a elementos com uma determinada classe. Exemplo: .classe
  • Seletor de ID: Aplica o estilo a um único elemento com um ID específico. Exemplo: #id

12. Grid no CSS

O CSS Grid é uma ferramenta poderosa para criar layouts bidimensionais na web. Ele permite organizar elementos em linhas e colunas de forma simples e eficaz. A sintaxe básica envolve declarar um contêiner grid e definir o número de colunas e linhas.

.grid-container {
            display: grid;
            grid-template-columns: 1fr 1fr 1fr;
            grid-gap: 10px;
        }

Os itens dentro do contêiner grid são organizados automaticamente em colunas. Para posicionar elementos individualmente:

.grid-item {
            grid-column: 1 / 3;
            grid-row: 1 / 2;
        }

Esse exemplo posiciona o item na primeira e segunda coluna e na primeira linha. O CSS Grid facilita a criação de layouts complexos com controle preciso sobre os elementos.
